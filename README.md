# Sistema de Gestão de Processos Administrativos Disciplinares

Sistema dedicado ao gerenciamento de PADs e Sindicâncias, permitindo o manejo de processos, usuários (internos e externos) e o envio de notificações acerca de prazos vincendos.

Para a instalação da aplicação, deverá ser criado um banco de dados MySQL com o nome indicado no arquivo *application.properties* (se outro banco for utilizado, as dependências deverão ser alteradas no arquivo *pom.xml*) e deverá ser inserido pelo menos um administrador antes de o sistema ser utilizado.
